## Technologies utilisées 
  - Angular 22.0.1 - Bootstrap 5.3 - TypeScript 5.9+ - npm 11.17.0 
  
## Fonctionnalités 
  - ✅ Layout responsive (Desktop / Tablette / Mobile) - ✅ Menu burger mobile - ✅ Carte interactive des plats avec catégories - ✅ Témoignages clients avec étoiles dynamiques - ✅ Horaires avec indicateur du jour ouvert - ✅ Footer multi-colonnes 
  
## Lancer le projet en local 
  
```bash 
git clone https://github.com/[VOTRE_USERNAME]/le-calao-dore.git 
cd le-calao-dore 
npm install 
ng serve --port 8080 --open 
``` 
  
Ouvrir [http://localhost:8080](http://localhost:8080).
